一个基于React + Next.js搭建的全栈笔记平台，轻松记录你的学习工作中的笔记。朋友也能够通过您的分享连接对您的笔记进行访问。

[在线Demo](https://yinp.cc/)

### <div style='padding:8px 0; border-bottom:2px solid #eee'>✨特性</div>
- 🌓暗黑模式
- 📱响应式能力
- :globe_with_meridians:笔记公开发布
- 🔒支持第三方登陆
- 📋Markdown语法支持
- 💎优雅美观
